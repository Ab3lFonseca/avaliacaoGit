Comandos mais utilizados no Git

git initInicializa um repositório Git
git clone [url]Clona um repositório remoto
git statusMostra o estado atual dos arquivos
git add .Adiciona todos os arquivos para o commit
git commit -m "mensagem"Registra alterações
git pushEnvia as alterações para o repositório remoto
git pullAtualiza o repositório local
git logExibe o histórico de commits
